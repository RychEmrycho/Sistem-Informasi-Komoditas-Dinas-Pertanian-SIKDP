# Sistem-Informasi-Komoditas-Dinas-Pertanian-SIKDP
=================================================================================

A Spring Boot Project {Thymeleaf, JPA, MySql | HSQL, Spring MVC, Spring Security} 

Main features: 
- Authentication (registration, login) 
- Authorization, Basic CRUD, File Uploading
